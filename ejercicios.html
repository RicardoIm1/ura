50 Ejercicios de Álgebra y Cálculo Básico

Ejercicio 1: Ecuación lineal

Resuelve: 3x + 7 = 16

Paso 1: Restamos 7 a ambos lados: 3x = 16 - 7

Paso 2: Simplificamos: 3x = 9

Paso 3: Dividimos entre 3: x = 3

Solución: x = 3

Aplicación práctica: Si un producto cuesta $7 más 3 veces su precio base, y el total es $16, este cálculo determina el precio base del producto.

Ejercicio 2: Ecuación cuadrática

Resuelve: x² - 5x + 6 = 0

Paso 1: Factorizamos: (x - 2)(x - 3) = 0

Paso 2: Igualamos cada factor a cero: x - 2 = 0 o x - 3 = 0

Paso 3: Soluciones: x = 2, x = 3

Soluciones: x = 2, x = 3

Aplicación práctica: En física, estas ecuaciones modelan trayectorias parabólicas. Por ejemplo, calcular a qué distancia caerá un objeto lanzado.

Ejercicio 3: Sistema de ecuaciones

Resuelve: 2x + y = 8 x - y = 1

Paso 1: Sumamos ambas ecuaciones: 3x = 9

Paso 2: Despejamos x: x = 3

Paso 3: Sustituimos en la segunda ecuación: 3 - y = 1 → y = 2

Solución: x = 3, y = 2

Aplicación práctica: Determinar precios cuando compras dos productos en diferentes combinaciones. Por ejemplo, si 2 hamburguesas y 1 refresco cuestan $8, y 1 hamburguesa menos 1 refresco cuesta $1.

Ejercicio 4: Derivada básica

Calcula la derivada de: f(x) = 3x² + 2x - 5

Paso 1: Aplicamos regla de potencia: d/dx(xⁿ) = n·xⁿ⁻¹

Paso 2: f'(x) = 2·3x²⁻¹ + 1·2x¹⁻¹ + 0

Paso 3: Simplificamos: f'(x) = 6x + 2

Solución: f'(x) = 6x + 2

Aplicación práctica: La derivada representa la tasa de cambio instantánea. En economía, indica cómo cambia el costo marginal al producir una unidad adicional.

Ejercicio 5: Integral indefinida

Calcula: ∫(4x³ + 2x) dx

Paso 1: Aplicamos regla de potencia inversa: ∫xⁿ dx = xⁿ⁺¹/(n+1)

Paso 2: ∫4x³ dx = 4x⁴/4 = x⁴

Paso 3: ∫2x dx = 2x²/2 = x²

Paso 4: Sumamos y añadimos constante: x⁴ + x² + C

Solución: x⁴ + x² + C

Aplicación práctica: La integral calcula el área bajo una curva. En negocios, permite determinar el ingreso total acumulado a partir de una función de ingreso marginal.

Ejercicio 6: Límite básico

Calcula: lim(x→2) (x² - 4)/(x - 2)

Paso 1: Factorizamos numerador: (x-2)(x+2)/(x-2)

Paso 2: Simplificamos: x + 2

Paso 3: Evaluamos en x=2: 2 + 2 = 4

Solución: 4

Aplicación práctica: Los límites determinan valores aproximados cuando una variable se acerca a un punto. En ingeniería, calculan resistencia máxima antes de la fractura.

Ejercicio 7: Logaritmos

Resuelve: log₂(x + 3) = 4

Paso 1: Aplicamos definición: 2⁴ = x + 3

Paso 2: Calculamos: 16 = x + 3

Paso 3: Despejamos x: x = 13

Solución: x = 13

Aplicación práctica: Los logaritmos miden la magnitud de fenómenos. En sismología, la escala Richter usa logaritmos para cuantificar la energía liberada en terremotos.

Ejercicio 8: Exponentes

Simplifica: (2³ · 2⁵) / 2⁴

Paso 1: Aplicamos propiedad: aᵐ · aⁿ = aᵐ⁺ⁿ → 2³⁺⁵ = 2⁸

Paso 2: Aplicamos propiedad: aᵐ / aⁿ = aᵐ⁻ⁿ → 2⁸ / 2⁴ = 2⁸⁻⁴

Paso 3: Simplificamos: 2⁴ = 16

Solución: 16

Aplicación práctica: Los exponentes modelan crecimiento exponencial. En finanzas, calculan interés compuesto: si inviertes $1000 al 5% anual, en 3 años tendrás 1000·(1.05)³.

Ejercicio 9: Fracciones algebraicas

Simplifica: (x² - 9)/(x² - 4x + 3)

Paso 1: Factorizamos numerador: (x-3)(x+3)

Paso 2: Factorizamos denominador: (x-1)(x-3)

Paso 3: Simplificamos: (x+3)/(x-1)

Solución: (x+3)/(x-1)

Aplicación práctica: Simplificar expresiones complejas es esencial en ingeniería para analizar circuitos eléctricos y determinar relaciones entre voltaje y corriente.

Ejercicio 10: Regla de la cadena

Deriva: f(x) = (3x² + 1)⁴

Paso 1: Identificamos función externa e interna: u = 3x² + 1, f(u) = u⁴

Paso 2: Aplicamos regla de la cadena: f'(x) = 4u³ · u'

Paso 3: Calculamos u': u' = 6x

Paso 4: Sustituimos: f'(x) = 4(3x² + 1)³ · 6x = 24x(3x² + 1)³

Solución: f'(x) = 24x(3x² + 1)³

Aplicación práctica: La regla de la cadena es fundamental en física para calcular cómo cambia la energía cinética respecto al tiempo cuando la velocidad también cambia.
