Ejercicios de Diagramas de Flujo

Practica y domina la representación gráfica de algoritmos

🔢 Ejercicio 1: Número Par o Impar

Fácil

Crear un diagrama de flujo que determine si un número ingresado por el usuario es par o impar.

📊 Ejercicio 2: Calcular Promedio

Fácil

Diseñar un diagrama que calcule el promedio de 3 calificaciones y determine si el alumno aprobó.

🔍 Ejercicio 3: Número Mayor

Medio

Elaborar un diagrama que encuentre el número mayor entre tres valores diferentes ingresados por el usuario.

🔄 Ejercicio 4: Serie Fibonacci

Medio

Crear un diagrama que genere los primeros N números de la serie Fibonacci.

🎯 Ejercicio 5: Adivina el Número

Difícil

Diseñar un diagrama para un juego donde el usuario debe adivinar un número aleatorio entre 1 y 100.

📈 Ejercicio 6: Estadísticas Básicas

Difícil

Elaborar un diagrama que calcule el promedio, máximo y mínimo de una lista de números.
