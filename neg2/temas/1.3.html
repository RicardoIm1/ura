• 1.3 Operaciones con matrices

    Objetivo del tema

    Comprender y aplicar las operaciones básicas entre matrices: suma, resta, multiplicación por un escalar y multiplicación entre matrices. El estudiante será capaz de identificar las dimensiones requeridas para cada operación, ejecutar los cálculos con precisión y reconocer situaciones donde estas operaciones son utilizadas en contextos prácticos como computación, economía e ingeniería.

    Idea principal

    Las matrices son arreglos rectangulares de números que se pueden combinar mediante reglas específicas. La suma y resta requieren que las matrices tengan exactamente las mismas dimensiones, mientras que la multiplicación de matrices exige que el número de columnas de la primera coincida con el número de filas de la segunda. Estas operaciones no son conmutativas y tienen propiedades algebraicas que extienden el álgebra de números reales, pero con diferencias clave.

    Ejercicios visualmente claros

    A continuación se presentan dos ejercicios resueltos paso a paso. Las matrices se muestran en formato visual y se incluyen gráficas que ayudan a interpretar la multiplicación como combinación lineal.

    Ejercicio 1: Suma y multiplicación escalar

    Dadas las matrices A y B, calcular 2A + B.

    A = 2 -1 0
    3 4 -2
    B = 1 3 4
    -2 0 1

    Paso a paso:

    • 1. Calcular 2A: se multiplica cada elemento de A por 2.
    • 2A = [[4, -2, 0], [6, 8, -4]]
    • 2. Sumar 2A + B: se suman elementos en la misma posición.
    • [[4+1, -2+3, 0+4], [6+(-2), 8+0, -4+1]] = [[5, 1, 4], [4, 8, -3]]
    Ejercicio 2: Multiplicación de matrices (2x3 por 3x2)

    Sean C (2x3) y D (3x2). El resultado será una matriz de dimensión 2x2.

    C = 1 0 2
    -1 3 1
    D = 2 1
    0 3
    4 -1

    Procedimiento comentado:

    // Elemento (1,1) = fila1 de C · col1 de D = (1*2)+(0*0)+(2*4) = 2+0+8 = 10
// Elemento (1,2) = fila1 de C · col2 de D = (1*1)+(0*3)+(2*-1) = 1+0-2 = -1
// Elemento (2,1) = fila2 de C · col1 de D = (-1*2)+(3*0)+(1*4) = -2+0+4 = 2
// Elemento (2,2) = fila2 de C · col2 de D = (-1*1)+(3*3)+(1*-1) = -1+9-1 = 7
// Resultado: C x D = [[10, -1], [2, 7]]

    Ejemplos de aplicaciones reales y directas

    • Computación gráfica: Las transformaciones geométricas (rotación, escalado, traslación) se representan con matrices. Multiplicar la matriz de transformación por las coordenadas de los vértices de un objeto produce su nueva posición.
    • Economía: En el análisis input-output de Leontief, las operaciones con matrices permiten calcular la producción necesaria en cada sector para satisfacer la demanda final.
    • Ingeniería de control: Los sistemas dinámicos se modelan con ecuaciones de estado en forma matricial, y las operaciones como suma y multiplicación se usan para combinar subsistemas.
    • Redes y grafos: La multiplicación de la matriz de adyacencia de un grafo cuenta caminos entre nodos, fundamental en algoritmos de rutas y recomendaciones.
