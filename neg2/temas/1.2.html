Clasificación de Matrices

TipoDescripciónEjemplo
SEGÚN SU DIMENSIÓN
Matriz Fila Tiene una sola fila [1 2 3] (1×3)
Matriz Columna Tiene una sola columna [1; 2; 3] (3×1)
Matriz Rectangular filas ≠ columnas 2×3, 3×2, 4×3
Matriz Cuadrada filas = columnas (n×n) 2×2, 3×3, 4×4
MATRICES CUADRADAS ESPECIALES
Triangular Superior ceros debajo de diagonal [a b c; 0 d e; 0 0 f]
Triangular Inferior ceros encima de diagonal [a 0 0; b c 0; d e f]
Matriz Diagonal solo diagonal no nula [a 0 0; 0 b 0; 0 0 c]
Matriz Escalar diagonal con el mismo valor [k 0 0; 0 k 0; 0 0 k]
Matriz Identidad unos en diagonal principal [1 0 0; 0 1 0; 0 0 1]
SEGÚN SUS ELEMENTOS
Matriz Nula (Cero) todos los elementos son cero [0 0 0; 0 0 0; 0 0 0]
Matriz Simétrica A = Aᵀ [a b c; b d e; c e f]
Matriz Antisimétrica A = -Aᵀ (diag. ceros) [0 a b; -a 0 c; -b -c 0]
POR OPERACIONES ESPECIALES
Matriz Traspuesta (Aᵀ) filas por columnas si A es 2×3, Aᵀ es 3×2
Matriz Regular (Inversible) existe A⁻¹, det ≠ 0 A·A⁻¹ = I
Matriz Singular no tiene inversa, det = 0 no existe A⁻¹
Matriz Ortogonal A·Aᵀ = Aᵀ·A = I A⁻¹ = Aᵀ
Matriz Idempotente A² = A A·A = A
Matriz Involutiva A² = I su cuadrado es I
