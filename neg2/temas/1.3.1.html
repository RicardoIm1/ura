• 1.3.1 Suma y resta de matrices

    Objetivo del tema

    El estudiante aprenderá a sumar y restar matrices de forma rigurosa, comprendiendo la condición de igualdad de dimensiones y la operación elemento a elemento. Se buscará que el alumno interprete estas operaciones como extensiones naturales de la aritmética escalar, adquiriendo soltura en el álgebra lineal básica.

    Idea principal

    La suma y resta de matrices se define exclusivamente para matrices del mismo orden (m × n). El resultado es otra matriz de la misma dimensión, cuyos elementos se obtienen sumando o restando los correspondientes elementos de las matrices originales. Esta operación hereda propiedades conmutativa, asociativa y elemento neutro (matriz cero) del álgebra de números reales, pero aplicada de manera estructurada.

    Visualización del concepto: adición matricial

    Qué notar: La matriz C conserva la disposición. Cada celda de C se obtiene de la suma de las celdas en la misma posición de A y B. Esto sólo tiene sentido si A y B tienen la misma cantidad de filas y columnas.

    Procedimiento paso a paso: suma de matrices 2×3

    /* Ejemplo numérico: A + B = C */
A = | 2  -1   3 |    B = |  0   5  -2 |
    | 4   0  -5 |        | -3   2   7 |

Paso 1: Verificar dimensiones: A es 2×3, B es 2×3 → OK.

Paso 2: Sumar elemento (1,1): 2 + 0 = 2
Paso 3: Sumar elemento (1,2): -1 + 5 = 4
Paso 4: Sumar elemento (1,3): 3 + (-2) = 1
Paso 5: Sumar elemento (2,1): 4 + (-3) = 1
Paso 6: Sumar elemento (2,2): 0 + 2 = 2
Paso 7: Sumar elemento (2,3): -5 + 7 = 2

Resultado C = | 2  4  1 |
              | 1  2  2 |

/* Para la resta A - B, se realiza: a_ij - b_ij en cada posición */

    Ejercicio visual: Grilla comparativa suma vs resta

    Qué notar: La resta no es conmutativa. Ambas operaciones preservan la estructura de rejilla; visualmente los valores pueden cambiar de signo, magnitud, pero siempre en la misma posición.

    Ejemplos de aplicaciones reales y directas

    • Imágenes digitales: Una imagen en escala de grises se representa como una matriz de píxeles. Sumar matrices permite combinar exposiciones (fotografía HDR) o promediar imágenes para reducir ruido.
    • Grafos y redes: En teoría de grafos, la matriz de adyacencia de un grafo puede sumarse con otra para representar la superposición de dos redes (ejemplo: vuelos de dos aerolíneas en los mismos aeropuertos).
    • Finanzas: Sumar matrices de ingresos por producto y región consolida balances trimestrales; restar matrices compara presupuestos vs gastos reales.
    • Física: Composición de desplazamientos mediante vectores en coordenadas (cada columna puede ser un vector).

    Tabla resumen: propiedades de la suma de matrices

    Propiedad Expresión Observación
    Conmutativa A + B = B + A Se cumple siempre (mismas dimensiones).
    Asociativa (A+B)+C = A+(B+C) Agrupación no altera resultado.
    Elemento neutro A + 0 = A Matriz cero (todos sus elementos 0).
    Elemento opuesto A + (-A) = 0 Matriz con todos los elementos cambiados de signo.
