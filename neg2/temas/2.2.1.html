• Objetivo del tema

    Aplicar las propiedades algebraicas de los límites para calcular el límite de funciones polinomiales, racionales y con raíces. Se aprenderá a resolver indeterminaciones del tipo 0/0 mediante factorización, racionalización y simplificación, así como a utilizar límites laterales cuando sea necesario.

    Idea principal

    El cálculo de límites se basa en propiedades que permiten descomponer funciones complejas en partes más simples. Cuando la sustitución directa produce una forma indeterminada como 0/0, es necesario manipular algebraicamente la función (factorizar, racionalizar, simplificar) para eliminar la indeterminación y encontrar el valor del límite. Estas técnicas son fundamentales para el estudio de la continuidad y la derivada.

    Ejercicios

    Ejercicio 1: Límite de una función polinomial (sustitución directa)

    Calcular lim_{x → 2} (3x² - 2x + 1). En funciones polinomiales, el límite se obtiene evaluando directamente.

    Paso Operación Resultado
    1Sustituir x = 2 en 3x² - 2x + 13(2)² - 2(2) + 1
    2Calcular potencias: (2)² = 43(4) - 4 + 1
    3Multiplicar: 3(4) = 1212 - 4 + 1
    4Sumar y restar9

    Comentario: Para funciones polinomiales, el límite cuando x → c es simplemente f(c). Esto se debe a que los polinomios son funciones continuas en todo ℝ.

    Ejercicio 2: Indeterminación 0/0 por factorización

    Calcular lim_{x → 2} (x² - 4)/(x - 2). La sustitución directa da 0/0. Factorizamos el numerador.

    Paso Operación Expresión
    1Sustitución directa(4 - 4)/(2 - 2) = 0/0 (indeterminado)
    2Factorizar numerador: x² - 4 = (x - 2)(x + 2)[(x - 2)(x + 2)]/(x - 2)
    3Simplificar (x - 2) (para x ≠ 2)x + 2
    4Evaluar el límite en la expresión simplificada2 + 2 = 4

    Qué notar en la gráfica: La función es una recta (y = x + 2) con un hueco en x = 2. El límite cuando x tiende a 2 es 4, que es el valor que tomaría la función si estuviera definida en ese punto. La gráfica confirma que el límite existe y vale 4.

    Ejercicio 3: Racionalización para eliminar indeterminación

    Calcular lim_{x → 0} (√(x+4) - 2)/x. Sustitución directa da 0/0. Racionalizamos el numerador.

    Paso Operación Expresión
    1Sustitución directa(√4 - 2)/0 = (2-2)/0 = 0/0
    2Multiplicar numerador y denominador por el conjugado: (√(x+4) + 2)[(√(x+4)-2)(√(x+4)+2)] / [x(√(x+4)+2)]
    3Simplificar numerador: (a-b)(a+b) = a² - b²[(x+4) - 4] / [x(√(x+4)+2)]
    4Simplificar: x / [x(√(x+4)+2)]1 / (√(x+4)+2) para x ≠ 0
    5Evaluar el límite (sustituir x = 0)1 / (√4 + 2) = 1/(2+2) = 1/4

    Qué notar en la gráfica: La función no está definida en x = 0 (hueco), pero se acerca al valor 0.25 desde ambos lados. La racionalización permite encontrar analíticamente este límite.

    Ejercicio 4: Límite con valor absoluto (límites laterales)

    Calcular lim_{x → 0} |x|/x. La función tiene comportamiento diferente por izquierda y derecha.

    Límite lateral Definición de |x| Simplificación Resultado
    x → 0⁻|x| = -x(-x)/x = -1-1
    x → 0⁺|x| = xx/x = 11

    Como los límites laterales son diferentes (-1 ≠ 1), el límite bilateral lim_{x → 0} |x|/x no existe.

    Qué notar en la gráfica: La función toma el valor -1 para x < 0 y 1 para x > 0. En x = 0 hay un salto, por lo que el límite bilateral no existe.

    Ejemplos de aplicaciones reales y directas

    • Física: velocidad instantánea. Al calcular la velocidad instantánea como límite de Δs/Δt cuando Δt → 0, frecuentemente aparecen indeterminaciones 0/0 que se resuelven simplificando la expresión de la posición.
    • Economía: costo marginal. El costo marginal C'(x) = lim_{h→0} [C(x+h)-C(x)]/h requiere técnicas de cálculo de límites cuando la función de costo es polinomial o racional.
    • Ingeniería: corriente en un capacitor. La relación i(t) = C dv/dt involucra límites de razones de cambio que pueden requerir racionalización en circuitos con elementos no lineales.
    • Biología: tasa de reacción enzimática. La cinética de Michaelis-Menten utiliza límites para determinar la velocidad máxima de una reacción.
