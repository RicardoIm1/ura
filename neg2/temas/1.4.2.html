• 1.4.2 Solución de un sistema de ecuaciones lineales por el método de Gauss-Jordan

    Tiempo estimado: 60 min

    Objetivo

    Aprender a resolver sistemas de ecuaciones lineales mediante el método de eliminación de Gauss-Jordan, transformando la matriz aumentada en su forma reducida por filas para obtener la solución de manera directa.

    Idea principal

    El método de Gauss-Jordan consiste en aplicar operaciones elementales sobre las filas de la matriz aumentada del sistema hasta alcanzar la forma escalonada reducida. En esta forma, cada variable tiene un coeficiente 1 en una única ecuación, lo que revela inmediatamente los valores de las incógnitas o la naturaleza del sistema (consistente, inconsistente o con infinitas soluciones).

    Ejercicio paso a paso: resolución de un sistema 2x2

    Sistema:
    2x + 3y = 8
    x - y = -1

    PasoOperaciónMatriz aumentada
    0Sistema inicial[ 2 3 | 8 ]
    [ 1 -1 | -1 ]
    1Intercambiar F1 ↔ F2 (para tener 1 como pivote)[ 1 -1 | -1 ]
    [ 2 3 | 8 ]
    2F2 = F2 - 2*F1[ 1 -1 | -1 ]
    [ 0 5 | 10 ]
    3F2 = (1/5)*F2[ 1 -1 | -1 ]
    [ 0 1 | 2 ]
    4F1 = F1 + F2[ 1 0 | 1 ]
    [ 0 1 | 2 ]

    Solución única: x = 1, y = 2.

    Gráfica del sistema: Las dos rectas se cortan en (1,2).

    Qué notar: La solución (x=1, y=2) es la intersección de ambas rectas; el método de Gauss-Jordan la obtiene directamente sin graficar, operando sobre las ecuaciones.

    Ejercicio 2: sistema 3x3

    x + 2y + z = 7
    2x - y + z = 4
    3x + y - 2z = -1

    PasoMatriz aumentada
    Inicio[1 2 1 | 7]
    [2 -1 1 | 4]
    [3 1 -2 | -1]
    F2 = F2 - 2F1
    F3 = F3 - 3F1    		[1 2 1 | 7]
    [0 -5 -1 | -10]
    [0 -5 -5 | -22]
    F3 = F3 - F2[1 2 1 | 7]
    [0 -5 -1 | -10]
    [0 0 -4 | -12]
    F3 = -1/4 F3[1 2 1 | 7]
    [0 -5 -1 | -10]
    [0 0 1 | 3]
    F2 = F2 + F3
    F1 = F1 - F3    		[1 2 0 | 4]
    [0 -5 0 | -7]
    [0 0 1 | 3]
    F2 = -1/5 F2[1 2 0 | 4]
    [0 1 0 | 7/5=1.4]
    [0 0 1 | 3]
    F1 = F1 - 2F2[1 0 0 | 4 - 2.8 = 1.2]
    [0 1 0 | 1.4]
    [0 0 1 | 3]

    Solución única: x = 1.2 , y = 1.4 , z = 3.

    En 3D las tres planos se cortan en un solo punto; el método nos entrega sus coordenadas mediante reducción completa.

    Aplicaciones reales y directas

    • Ingeniería estructural: Cálculo de fuerzas internas en armaduras, donde cada nodo genera una ecuación lineal de equilibrio.
    • Circuitos eléctricos: Análisis de mallas y nodos mediante leyes de Kirchhoff producen sistemas lineales resueltos con Gauss-Jordan.
    • Economía: Modelos input-output de Leontief, que describen relaciones entre sectores productivos, se resuelven con eliminación gaussiana.
    • Química: Balanceo de reacciones químicas: se igualan cantidades de átomos planteando ecuaciones lineales.
