Mi Primer Función

Paso 1: Revisa que tu código sea como este:

function calculadora(a, b, operacion) {
    switch(operacion) {
        case '+':
            return a + b;
        case '-':
            return a - b;
        case '*':
            return a * b;
        case '/':
            return b !== 0 ? a / b : "Error: División por cero";
        default:
            return "Operación no válida";
    }
}

// Uso:
console.log(calculadora(10, 5, '+'));  // 15
console.log(calculadora(10, 5, '-'));  // 5
console.log(calculadora(10, 5, '*'));  // 50
console.log(calculadora(10, 5, '/'));  // 2
console.log(calculadora(10, 0, '/'));  // Error: División por cero

Paso 2: Explicación del código

Este código define una función llamada calculadora que toma tres parámetros: a, b y operacion. Dependiendo del valor de operacion, la función realiza una de las cuatro operaciones matemáticas básicas (suma, resta, multiplicación o división) entre a y b. Si la operación es una división, también verifica que b no sea cero para evitar errores. Si la operación no es válida, devuelve un mensaje de error.

Luego, se muestran ejemplos de cómo usar la función calculadora con diferentes operaciones y se imprimen los resultados en la consola.

Paso 3: Prueba el código

Puedes copiar y pegar este código en la consola de tu navegador web o en un entorno de desarrollo de JavaScript para ver cómo funciona. Modifica los valores de a, b y operacion para experimentar con diferentes cálculos.
