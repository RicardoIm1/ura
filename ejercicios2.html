Ejercicio 1: Ecuación lineal

Resuelve: 3x + 7 = 16

Paso 1: Restamos 7 a ambos lados: 3x = 16 - 7

Paso 2: Simplificamos: 3x = 9

Paso 3: Dividimos entre 3: x = 3

Solución: x = 3

Aplicación práctica: Si un producto cuesta $7 más 3 veces su precio base, y el total es $16, este cálculo determina el precio base del producto.

Ejercicio 2: Ecuación cuadrática

Resuelve: x² - 5x + 6 = 0

Paso 1: Factorizamos: (x - 2)(x - 3) = 0

Paso 2: Igualamos cada factor a cero: x - 2 = 0 o x - 3 = 0

Paso 3: Soluciones: x = 2, x = 3

Soluciones: x = 2, x = 3

Aplicación práctica: En física, estas ecuaciones modelan trayectorias parabólicas. Por ejemplo, calcular a qué distancia caerá un objeto lanzado.

Ejercicio 3: Sistema de ecuaciones

Resuelve: 2x + y = 8
x - y = 1

Paso 1: Sumamos ambas ecuaciones: 3x = 9

Paso 2: Despejamos x: x = 3

Paso 3: Sustituimos en la segunda ecuación: 3 - y = 1 → y = 2

Solución: x = 3, y = 2

Aplicación práctica: Determinar precios cuando compras dos productos en diferentes combinaciones.

Ejercicio 4: Derivada básica

Calcula la derivada de: f(x) = 3x² + 2x - 5

Paso 1: Aplicamos regla de potencia: d/dx(xⁿ) = n·xⁿ⁻¹

Paso 2: f'(x) = 2·3x²⁻¹ + 1·2x¹⁻¹ + 0

Paso 3: Simplificamos: f'(x) = 6x + 2

Solución: f'(x) = 6x + 2

Aplicación práctica: La derivada representa la tasa de cambio instantánea. En economía, indica cómo cambia el costo marginal.

Ejercicio 5: Integral indefinida

Calcula: ∫(4x³ + 2x) dx

Paso 1: Aplicamos regla de potencia inversa: ∫xⁿ dx = xⁿ⁺¹/(n+1)

Paso 2: ∫4x³ dx = 4x⁴/4 = x⁴

Paso 3: ∫2x dx = 2x²/2 = x²

Paso 4: Sumamos y añadimos constante: x⁴ + x² + C

Solución: x⁴ + x² + C

Aplicación práctica: La integral calcula el área bajo una curva. En negocios, permite determinar el ingreso total acumulado.

Paso 2: Simplificamos: x + 2

Paso 3: Evaluamos en x=2: 2 + 2 = 4

Solución: 4

Aplicación práctica: Los límites determinan valores aproximados cuando una variable se acerca a un punto.

Ejercicio 8: Exponentes

Simplifica: (2³ · 2⁵) / 2⁴

Paso 1: Aplicamos propiedad: aᵐ · aⁿ = aᵐ⁺ⁿ → 2³⁺⁵ = 2⁸

Paso 2: Aplicamos propiedad: aᵐ / aⁿ = aᵐ⁻ⁿ → 2⁸ / 2⁴ = 2⁸⁻⁴

Ejercicio 10: Regla de la cadena

Deriva: f(x) = (3x² + 1)⁴

Paso 1: Identificamos función externa e interna: u = 3x² + 1, f(u) = u⁴

Paso 2: Aplicamos regla de la cadena: f'(x) = 4u³ · u'

Paso 3: Calculamos u': u' = 6x

Paso 4: Sustituimos: f'(x) = 4(3x² + 1)³ · 6x = 24x(3x² + 1)³

Solución: f'(x) = 24x(3x² + 1)³

Aplicación práctica: La regla de la cadena es fundamental en física para calcular cómo cambia la energía cinética respecto al tiempo.

Ejercicio 18: Regla del producto

Deriva: f(x) = (x² + 1)(3x - 2)

Paso 1: Identificamos u = x² + 1, v = 3x - 2

Paso 2: u' = 2x, v' = 3

Paso 3: Aplicamos regla del producto: f'(x) = u'v + uv'

Paso 4: f'(x) = (2x)(3x-2) + (x²+1)(3) = 6x² - 4x + 3x² + 3 = 9x² - 4x + 3

Solución: f'(x) = 9x² - 4x + 3

Aplicación práctica: En ingeniería, calcular cómo cambia el volumen de un cilindro cuando tanto el radio como la altura cambian simultáneamente.

Ejercicio 19: Regla del cociente

Deriva: f(x) = (x² + 1)/(2x - 1)

Paso 1: Identificamos u = x² + 1, v = 2x - 1

Paso 2: u' = 2x, v' = 2

Paso 3: Aplicamos regla del cociente: f'(x) = (u'v - uv')/v²

Paso 4: f'(x) = [(2x)(2x-1) - (x²+1)(2)]/(2x-1)² = [4x²-2x-2x²-2]/(2x-1)² = (2x²-2x-2)/(2x-1)²

Solución: f'(x) = (2x²-2x-2)/(2x-1)²

Aplicación práctica: En economía, calcular la elasticidad precio de la demanda, que es el cociente entre cambio porcentual en cantidad y cambio porcentual en precio.

Ejercicio 48: Ecuación de Bernoulli

Resuelve: y' + y = xy²

Paso 1: Dividimos entre y²: y'/y² + 1/y = x

Paso 2: Hacemos u = 1/y, entonces u' = -y'/y²

Paso 3: Sustituimos: -u' + u = x → u' - u = -x

Paso 4: Factor integrante: e^(-x)

Paso 5: Solución: u = x + 1 + Ce^x → y = 1/(x + 1 + Ce^x)

Solución: y = 1/(x + 1 + Ce^x)

Aplicación práctica: En dinámica de fluidos, modelar el flujo de líquidos no newtonianos donde la viscosidad depende de la velocidad de corte.

Ejercicio 49: Función multivariable

Derivadas parciales de f(x,y)=x²y + sen(xy)

Paso 1: ∂f/∂x: tratamos y como constante

Paso 2: ∂f/∂x = 2xy + y·cos(xy)

Paso 3: ∂f/∂y: tratamos x como constante

Paso 4: ∂f/∂y = x² + x·cos(xy)

Solución: ∂f/∂x = 2xy + y·cos(xy), ∂f/∂y = x² + x·cos(xy)

Aplicación práctica: En economía, calcular cómo cambia la producción cuando varían simultáneamente los insumos capital y trabajo.

Ejercicio 50: Gradiente y derivada direccional

Para f(x,y)=x²+y², encuentra ∇f(1,2) y Dᵤf(1,2) donde u=(3/5,4/5)

Paso 1: ∇f = (∂f/∂x, ∂f/∂y) = (2x, 2y)

Paso 2: ∇f(1,2) = (2, 4)

Paso 3: Dᵤf = ∇f · u

Paso 4: Dᵤf(1,2) = (2,4)·(3/5,4/5) = 6/5 + 16/5 = 22/5

Solución: ∇f(1,2) = (2,4), Dᵤf(1,2) = 22/5

Aplicación práctica: En machine learning, el gradiente indica la dirección de máximo crecimiento para optimizar funciones de costo durante el entrenamiento de modelos.
