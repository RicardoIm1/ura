2. DISEÑO DE ALGORITMOS

2.1 Conceptos básicos

Para empezar a diseñar algoritmos, debemos tener claros los cimientos. Imagina que un algoritmo es como una receta de cocina...

Definición formal: Un algoritmo es un conjunto ordenado y finito de operaciones simples y bien definidas...

  • Preciso: Cada instrucción debe ser clara...
  • Definido: Si se sigue un algoritmo dos veces...
  • Finito: Debe tener un número limitado de pasos...
  • Entrada (Input): Los datos que necesita el algoritmo...
  • Proceso: Las operaciones y cálculos...
  • Salida (Output): El resultado final obtenido...

2.2 Metodología de la programación

  1. Análisis del problema: Entender qué se necesita resolver...
  2. Diseño del algoritmo: Definir cómo lo vamos a resolver...
  3. Codificación: Traducir el algoritmo a un lenguaje...
  4. Prueba y depuración: Ejecutar el programa...
  5. Mantenimiento: Mejorar el programa...

2.3 Diagramas de flujo

  • Óvalo: Inicio o fin.
  • Paralelogramo: Entrada/Salida.
  • Rectángulo: Proceso.
  • Rombo: Decisión.
  • Flechas: Dirección del flujo.

2.4 Seudocódigos

Algoritmo NombreDelAlgoritmo
    Definir variable1, variable2 Como TipoDeDato
    Escribir "Mensaje"
    Leer variable1
    variable2 <- variable1 * 2
    Si (condicion) Entonces
        ...
    FinSi
    Escribir "Resultado: ", variable2
FinAlgoritmo

Ejemplo 1: Área de rectángulo

Algoritmo AreaRectangulo
    Definir base, altura, area Como Real
    Leer base, altura
    area <- base * altura
    Escribir "Área: ", area
FinAlgoritmo

Ejemplo 2: Par o impar

Algoritmo ParImpar
    Definir numero Como Entero
    Leer numero
    Si (numero MOD 2 == 0) Entonces
        Escribir "PAR"
    Sino
        Escribir "IMPAR"
    FinSi
FinAlgoritmo

Ejemplo 3: Suma hasta N

Algoritmo SumaHastaN
    Definir n, i, suma Como Entero
    Leer n
    i <- 1
    suma <- 0
    Mientras (i <= n) Hacer
        suma <- suma + i
        i <- i + 1
    FinMientras
    Escribir "Suma: ", suma
FinAlgoritmo

3.9 Ejemplo completo: Calculadora

#include <stdio.h>
int main() {
    float num1, num2, resultado;
    int opcion;
    scanf("%f", &num1);
    scanf("%f", &num2);
    if(opcion == 1) resultado = num1 + num2;
    else if(opcion == 2) resultado = num1 - num2;
    else if(opcion == 3) resultado = num1 * num2;
    else if(opcion == 4) {
        if(num2 != 0) resultado = num1 / num2;
        else printf("Error: división entre cero");
    }
    return 0;
}
