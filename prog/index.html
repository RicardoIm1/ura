Objetivo General

Prácticas de laboratorio

Realiza prácticas de laboratorio de programación en el lenguaje C.

  • Requisitos:

    • Abrir una cuenta en GitHub.io
    • Abrir una cuenta en DeepSeek.io
    • Deberás generar el código a través de un prompt y hospedarlo en linea para que sea de acceso público
    • Debe realizar operaciones básicas (+, -, ×, ÷)
    • Debe incluir funciones científicas (seno, coseno, tangente, raíz cuadrada, potencia)
    • Debe tener cambio de tema (claro/oscuro)
    • Diseño responsive (adaptable a móviles)
    • Validación de errores (división entre cero, expresiones inválidas)
    • Entregar código comentado y documentado

  • Requisitos:

    • Datos personales
    • Formación académica (escuela, período, título obtenido)
    • Experiencia laboral (puesto, empresa, fechas, funciones)
    • Habilidades y competencias (idiomas, software, certificaciones)
    • Diseño profesional y atractivo (usar CSS moderno)
    • Deberás generar el código a través de un prompt y hospedarlo en linea para que sea de acceso público
    • Maquetado con HTML semántico y CSS Grid/Flexbox
