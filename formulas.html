Álgebra Básica Ley de Signos (Multiplicación/División) (+) * (+) = (+) (-) * (-) = (+) (+) * (-) = (-) Aplicación: Fundamental para todos los cálculos contables y financieros. Determina si una ganancia (positivo) o una pérdida (negativo) resulta de combinar ingresos y costos.
Álgebra Básica Propiedad Distributiva a(b + c) = ab + ac Aplicación: Simplificar expresiones complejas en presupuestos y costos. Ejemplo: Calcular el costo total de varios artículos con el mismo impuesto.
Álgebra Básica Despeje de una Variable ax + b = c → x = (c - b)/a Aplicación: Encontrar el valor de una variable desconocida crítica, como el precio unitario necesario para alcanzar un ingreso total deseado (Ingresos = Precio * Cantidad).
Álgebra - Productos Notables Binomio al Cuadrado (a + b)² = a² + 2ab + b² (a - b)² = a² - 2ab + b² Aplicación: Modelar el crecimiento no lineal. Ejemplo: Si el costo de mano de obra (a) y los materiales (b) aumentan, el costo total aumenta de forma cuadrática, no lineal.
Álgebra - Productos Notables Binomios Conjugados (a + b)(a - b) = a² - b² Aplicación: Simplificar cálculos. Ejemplo: Calcular la diferencia de áreas o, en finanzas, la diferencia entre el valor futuro al cuadrado de dos inversiones.
Ecuaciones Cuadráticas Fórmula General x = [-b ± √(b² - 4ac)] / (2a) (para ax² + bx + c = 0) Aplicación: Encontrar el punto de equilibrio en un negocio donde los costos e ingresos tienen una relación cuadrática. Las soluciones (raíces) representan los niveles de producción donde la utilidad es cero.
Ecuaciones Cuadráticas Vértice de una Parábola Vx = -b / (2a) Vy = f(Vx) Aplicación: Encontrar el punto de máxima utilidad o mínimo costo. Vx es el nivel de producción óptimo y Vy es la utilidad máxima o costo mínimo alcanzable. Es crucial para la optimización.
Sucesiones y Series Sucesión Aritmética (término n-ésimo) aₙ = a₁ + (n - 1)d Aplicación: Modelar crecimientos o decrementos lineales. Ejemplo: Proyectar el salario de un empleado que recibe un aumento fijo anual (d) cada año.
Sucesiones y Series Suma de una Serie Aritmética Sₙ = n/2 * (a₁ + aₙ) ó Sₙ = n/2 * [2a₁ + (n - 1)d] Aplicación: Calcular el total acumulado en esos escenarios lineales. Ejemplo: Calcular el total pagado en salarios durante n años.
Sucesiones y Series Sucesión Geométrica (término n-ésimo) aₙ = a₁ * r⁽ⁿ⁻¹⁾ Aplicación: Modelar crecimientos o decrementos exponenciales (por porcentajes). Ejemplo: Proyectar el valor de una inversión que crece a una tasa de interés constante (r), o la depreciación de un activo.
Sucesiones y Series Suma de una Serie Geométrica Sₙ = a₁ * (1 - rⁿ) / (1 - r) Aplicación: Calcular el total acumulado en un periodo con crecimiento porcentual. Ejemplo: Sumar todos los ingresos de una patente que genera regalías que disminuyen un porcentaje fijo cada año.
Sucesiones y Series Suma de Serie Geométrica Infinita S = a₁ / (1 - r) (si |r| < 1) Aplicación: Calcular el valor presente de una perpetuidad. Ejemplo: Determinar el precio justo a pagar hoy por un activo que generará un flujo de ingreso constante (una renta) para siempre.
Matemáticas Financieras Interés Simple I = P * i * t M = P + I ó M = P(1 + i*t) Aplicación: Calcular el interés de una inversión o deuda a corto plazo donde el interés no se capitaliza. Común en préstamos informales o algunos tipos de inversiones.
Matemáticas Financieras Interés Compuesto (Valor Futuro) M = C * (1 + i)ⁿ Aplicación: La base de las finanzas modernas. Calcula el valor futuro de una inversión inicial (C) donde los intereses se reinvierten. Usado para planificación de ahorro, jubilación y evaluación de inversiones.
Matemáticas Financieras Valor Presente con Interés Compuesto C = M / (1 + i)ⁿ ó C = M * (1 + i)⁻ⁿ Aplicación: Descontar flujos futuros a valor presente. Responde: ¿Cuánto debo invertir hoy (C) para tener un monto objetivo (M) en el futuro? Es fundamental para valorar activos y proyectos.
Matemáticas Financieras Tasa de Interés Efectiva i_ef = (1 + i/m)^m - 1 Aplicación: Permite comparar diferentes opciones de inversión o crédito que tienen distintos periodos de capitalización (anual, semestral, mensual, etc.). Standardiza la tasa a un periodo anual común.
Matemáticas Financieras Anualidades Vencidas (Valor Futuro) M = R * [ ((1 + i)ⁿ - 1) / i ] Aplicación: Calcular el monto acumulado al hacer aportes periódicos y constantes (R) a un fondo de inversión (ej. plan de pensiones). Los pagos se hacen al final de cada periodo.
Matemáticas Financieras Anualidades Vencidas (Valor Presente) C = R * [ (1 - (1 + i)⁻ⁿ) / i ] Aplicación: Calcular el valor presente de una serie de pagos futuros. Responde: ¿Cuánto vale hoy un contrato que me pagará una renta fija (R) durante n periodos? Se usa para valorar pensiones, loterías y calcular mensualidades de créditos.
Matemáticas Financieras Anualidades Anticipadas (Valor Futuro) M = R * [ ((1 + i)ⁿ - 1) / i ] * (1 + i) Aplicación: Similar a la vencida, pero los pagos se hacen al inicio del periodo (ej. pagos de arriendo). Genera un monto futuro ligeramente mayor debido a que cada pago se invierte un periodo más.
Matemáticas Financieras Anualidades Anticipadas (Valor Presente) C = R * [ (1 - (1 + i)⁻ⁿ) / i ] * (1 + i) Aplicación: Calcular el valor presente cuando los pagos o ingresos son anticipados. Es el caso típico de los leasing o arrendamientos financieros.
Matemáticas Financieras Amortización (Saldo Insoluto) Saldo = VP de los pagos restantes Aplicación: Determinar cuánto se le debe aún al banco después de haber hecho k pagos de un crédito. Esencial para generar tablas de amortización y saber el estado de la deuda en cualquier momento.
Matemáticas Financieras Depreciación Lineal Depreciación Anual = (Costo - Valor de Salvamento) / Vida Útil Aplicación: Distribuir el costo de un activo tangible (maquinaria, vehículos) de forma lineal a lo largo de su vida útil. Afecta directamente al estado de resultados y al balance general.
Matemáticas Financieras Depreciación por Porcentaje Fijo Valor en k años = Costo * (1 - d)ᵏ Aplicación: Modelar la pérdida de valor de un activo que es más pronunciada en los primeros años (ejemplo: un vehículo nuevo). d es la tasa de depreciación anual.
